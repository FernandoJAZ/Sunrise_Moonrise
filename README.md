# SunriseSunset

Este script te permite extraer rápidamente los datos de la página actual en formato CSV y copiarlos al portapapeles. Tendrás que cambiar manualmente la URL para cada mes o abri el mes deseado en el menú de la propia web.
La url es:
https://www.timeanddate.com/sun/spain/almeria?month=1&year=2026
